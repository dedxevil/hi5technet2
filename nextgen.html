20-Day Curriculum

1

Phase 1: Flutter Flow Basics + Intro to Prompting

Day 1: What is Flutter, Flutter Flow, Dart?

Explain low-code vs full-code. Signup on Flutter Flow.

Day 2: Build UI in Flutter Flow

Create "Hello App" with Home and About pages.

Day 3: Navigation and Form Widgets

Add navigation between pages and a form (name, email).

Day 4: Prompt Engineering Basics

Use Chat GPT prompts to generate app ideas and dummy content.

Day 5: Project 1: Personal Diary App

Design UI and logic with Flutter Flow. Add a form to save entries.

2

Phase 2: Firebase, Custom Logic, and Intro to VS Code

Day 6: Firebase Setup in Flutter Flow

Add authentication (Login and Signup).

Day 7: Firestore Database

Save and display user notes or tasks in Firestore.

Day 8: Chat GPT + Custom Dart

Prompt Chat GPT for simple functions (e.g., calculate age) and add them in Flutter Flow.

Day 9: Flutter + VS Code Setup

Install Flutter SDK, VS Code, and create a "Hello World" project.

Day 10: Compare UI Building

Build the same screen in Flutter Flow and Flutter. Discuss pros and cons.

3

Phase 3: Code Logic + Flutter Widgets with Chat GPT

Day 11: VS Code: Layout Widgets

Use Scaffold, App Bar, Column, Row. Use Chat GPT to assist.

Day 12: State Management (setState)

Build a counter app with increment logic.

Day 13: List View Builder

Create a dynamic to-do list app.

Day 14: Chat GPT Challenge

Students use Chat GPT to build a custom widget (e.g., quiz, feedback form).

Day 15: Project 2: Notes App in Flutter

Build a local notes app using VS Code. Add, view, and delete notes.

4

Phase 4: Final Project + Integration + Showcase

Day 16: Build Final App UI in Flutter Flow

Group selects final project idea (quiz app, journal app, recipe app).

Day 17: Export Flutter Code from Flutter Flow

Open exported project in VS Code. Understand file structure.

Day 18: Modify Code with Chat GPT

Add new features or visual polish using prompt-generated code.

Day 19: Final Testing and Polish

Debug, test login, validate forms, and review UI consistency.

Day 20: Presentation and Demo Day

Students present their app, explain how Chat GPT helped, and receive feedback.
